# Слайд 2  
## Теоретическое введение  

- **НОД(a, b)** — наибольшее целое число, делящее оба числа без остатка.  
  Пример: НОД(12, 18) = 6  
- Если НОД = 1 → числа **взаимно простые** (например, 8 и 15)  
- **Применения**:  
  - Сокращение дробей  
  - Криптография (RSA)  
  - Решение диофантовых уравнений

# Слайд 4  
## Обзор алгоритмов  

1. **Классический Евклид** — деление с остатком  
2. **Бинарный Евклид** — только сдвиги и вычитания  
3. **Расширенный Евклид** — находит $x, y$: $a \cdot x + b \cdot y = \text{НОД}(a, b)$  
4. **Расширенный бинарный** — битовые операции + коэффициенты  

> Все алгоритмы дают одинаковый НОД, но различаются по реализации.

# Слайд 6  
## Демонстрация кода (расширенный алгоритм Евклида)  

```julia
function extended_gcd(a::Int, b::Int)::Tuple{Int, Int, Int}
    if b == 0
        return (abs(a), sign(a), 0)
    end
    x0, x1 = 1, 0
    y0, y1 = 0, 1
    a, b = abs(a), abs(b)
    while b != 0
        q = a ÷ b
        a, b = b, a % b
        x0, x1 = x1, x0 - q * x1
        y0, y1 = y1, y0 - q * y1
    end
    return (a, x0 * sign(a), y0 * sign(b))
end
```
